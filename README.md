# The Last Bus

The Last Bus is a 2D Unity-based game developed as an indie-style project focusing on atmosphere, storytelling, and gameplay mechanics.

---

** About the Game

Genre: 2D Indie / Horror / Adventure 

Engine: Unity

Platform: PC

Style: Dark, minimal, story-driven

---

** Features

2D gameplay mechanics

Custom level design

Character movement & interactions

Atmospheric environment and visuals

Dialogues and story elements

---

** Tech Stack

Unity Engine

C#

---

** Purpose of the Project

Learn Unity game development

Understand game mechanics & level design

Practice C# scripting in real projects

Build a complete playable game

---

** Note

This is an indie / learning project

Assets may be placeholders or self-designed

Game mechanics may evolve over time
